an extension + dashboard pair for teachers to keep an eye on student web activity.

overview

the extension was created to help teachers monitor student web activity. it allows teachers to see what students are doing on their computers, and the dashboard provides a simple interface for teachers to manage their students' web activity. The program is also easy to use, making it a great tool for teachers who may not be tech-savvy.

features

  • online/offline tracking
  • privacy-focused data tracking
  • secure info relay
  • lightweight extension

challenges

The main challenge was prioritizing user data security. Although this initially conflicted with some ideas for features, the issues were soon resolved.
