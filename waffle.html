A simple, secure chat app for students.

overview

The app was created as a test to replace other apps such as iMessage, Discord, and Slack. While it may be simpler, it allows students to communicate securely and efficiently and does not collect data.

features

  • Fast messaging
  • Privacy-focused framework
  • Secure login
  • Accessibility-first approach

challenges

The main challenge was balancing feature richness with user experience. By carefully ensuring I prioritized features my users wanted, I was able to maintain a lightweight application while still providing a great user experience.
